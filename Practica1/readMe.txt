Práctica 1

Para la práctica deberán realizar un “sitio” web con HTML, CSS y javascript.
Deberán generar un árbol de archivos de la siguiente manera.

+MiPagina
  -index.html
  +html
    -otraPagina.html
    -otraPagina.html
  +css
    -estilo1.css
    -estilo2.css
  +js
    -script1.js
    -script2.js

Los archivos html deberán incluir las hojas de estilo (css) y los archivos de script (js) en el header del archivo.

-Como mínimo deben incluir index.html y otro html, un archivo css y un archivo de javascript.
-HTML debe incluir: Video, audio, imágenes, formulario, listas y lo que se requiera para mostrar el
funcionamiento de css y javascript.
-CSS debe incluir: Estilo para por lo menos 2 etiquetas, 2 clases y 2 clases de una etiqueta.
-Javascript debe incluir: Uso de variables, arreglos, ciclos “for” y “while”, uso de “if”, uso de funciones con variables locales y globales, y una función para atender un formulario de html.

En cuanto a javascript no es necesario que las funciones y ciclos tengan sentido (excepto la función que
atiende el formulario) mientras hagan operaciones con las variables (operaciones, concatenaciones, etc)
